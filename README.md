# ChattingRoom
A simple and easy online chatting room built by JavaScript, Socket.io, MaterializeCss and ExpressJS,


## Getting Started
```
npm install 
```
```
node server.js
```
You should see "server running..." on your terminal.  Open: localhost:5000 and connect to socket
